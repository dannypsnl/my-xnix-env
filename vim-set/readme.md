# vim-set
This repo put my setting of vim.
It based on spf13
## Usage
```bash
$ git clone https://github.com/dannypsnl/my-xnix-env.git
$ mv vim-set/.vimrc.* ~/
$ mv vim-set/syntax/* ~/.vim/syntax/
$ . spf13-vim.sh
```
## Support
### languages
- those spf13 contain
- nim
- elm
- swift
- go-lang
- antlr's grammar
### tools
- google/codefmt
- prettier for JavaScript, React, css and html
### themes
- material for gvim only(true color, neovim)
