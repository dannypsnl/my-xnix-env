# git-alias-setting
## Initalize
* Use following command  
```bash
$ git clone https://github.com/dannypsnl/my-xnix-env.git
$ cd git-alias-setting/
$ sh ./git-alias-setting.sh
```
## API
### Checkout
`$ git co <branch>`
### Branch
`$ git br <--params>`
### Commit
`$ git cm <--params>`
### Merge
`$ git mg <branch>`
### Pretty Tree
`$ git tree`
### Push all(--mirror)
`$ git pushall`
### Add
`$ git ad <--path, files>`
### Diff
`$ git df <commit-log>`
### Status
`$ git st`
### Rebase
`$ git rb`
