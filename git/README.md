# git-alias-setting

### Alias

-  Checkout
    `$ git co <branch>`
-  Branch
    `$ git br <--params>`
-  Commit
    `$ git cm <--params>`
-  Merge
    `$ git mg <branch>`
-  Pretty Tree
    `$ git tree`
-  Add
    `$ git ad <--path, files>`
-  Diff
    `$ git df <commit-log>`
-  Status
    `$ git st`
-  Rebase
    `$ git rb`

### setup global ignore file

```
git config --global core.excludesfile ~/.gitignore_global
```
